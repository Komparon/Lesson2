## Списки

Чтобы добавить ненумерованные списки, необходимо выделить пункты звездочкой (*).
Например вот так:

* Элемент 1
* Элемент 2
* Элемент 3

Чтобы добавить нумерованные списки, необходимо пункты просто пронумеровать.
Например вот так:

1. Элемент 1
2. Элемент 2
3. Элемент 3
4. Элемент 4
